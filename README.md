# Alpha Browser

A custom, lightweight web browser built from scratch in **Python**, using **PyQt5** for the graphical interface.

## 📌 About

Alpha Browser is a hands-on exploration into how web browsers work under the hood — built using PyQt5's `QWebEngineView` to render web pages within a custom-built desktop application. The project focuses on implementing core browser functionality (navigation, address bar, tabs, etc.) using Python and Qt's GUI framework.

## 🛠️ Tech Stack

| Category | Tools |
|---|---|
| Language | Python |
| GUI Framework | PyQt5 |
| Web Rendering | Qt WebEngine (`QWebEngineView`) |

## ✨ Features

- Custom-built browser window using PyQt5
- Address bar for entering and navigating to URLs
- Basic browser controls (back, forward, reload, etc.)
- Web page rendering via Qt's WebEngine component

## 🚀 Getting Started

### Prerequisites
- Python 3.x
- PyQt5 and PyQtWebEngine:
```bash
pip install PyQt5 PyQtWebEngine
```

### Setup
```bash
git clone https://github.com/MiruthyanJayanS/alpha-browser-using-python.git
cd alpha-browser-using-python
pip install -r requirements.txt
```

### Running
```bash
python main.py
```
> Replace `main.py` with the actual entry-point filename used in the project.
